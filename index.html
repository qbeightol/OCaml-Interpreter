OCaml-Interpreter

An OCaml interpreter written in OCaml.

Written for Problem Set 4 of CS 3110 (spring 2014); part1 includes a problem on streams (which is completely unrelated to the interpreter), while part 2 includes the implementation of the interpreter. Check out assignment_description.pdf for a more detailed overview of the assignment.
